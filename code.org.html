Summary of Assignment & Requirements:

Complete the planning guide. Complete parts 3 - 12 in code.org. Once at part 12 you should have a completed game that meets the rubric criteria. Design and code a game in code.org.

Challenges & My Solution:

The assignment wanted me to create an entire game from scratch. Even though I had a planning document, I couldn't come up with any new game ideas. So I chose to recreate an existing game, Flappy Bird. While coding Flappy Bird, I ran into a few challenges.
Challenge #1 was how to get good looking sprites as I can't design good-looking ones. To solve this issue I simply looked up for Flappy Bird sprites.
Challenge #2 was trying to find a way to detect bird collisons with the pillars while also allowing the bird to fly throgh the pillars. To solve this, I just used two sprites, one was the top pillar and one was the the bottom pillar.
Challenge #3 was creating a restart screen. I wanted to do this because in code.org, to reset the game you must click the "Reset" button (not part of the program you code) and then click "Run" (not part of the program you code). Thing is, code.org has no "button" element like it does "Text" element. So I used onclick events with button sprites to recreate the same effect. I used this for a restart button.
